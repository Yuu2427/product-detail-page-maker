# Product Detail Page Maker 使用说明

## 1. 准备商品图片

将商品实拍图放入：

```text
商品实拍图/
```

图片名称建议包含商品名称。

## 2. 准备商品信息

填写根目录中的：

```text
商品详情页生成表.xlsx
```

重点填写 `商品信息_必填`。不知道的信息写“未知，不要编造”。

也可以先放好图片，再让 AI 帮你识别和补充表格。

## 3. 调用 Skill

```text
请使用 ecommerce-gpt-image2-pdp-high-fidelity skill，
为商品详情页生成表里的【商品名称】生成全套详情页。

我已经把商品实拍图放到对应文件夹。
请先检查商品信息，向我确认缺失内容，
并给出头图和六张详情页切片规划。

得到我确认前不要生成图片。
所有生图必须使用 gpt-image-2；如果该模型不可用，请停止并说明，
不要自动换用其他模型。
```

## 4. 确认并生成

1. 确认商品信息和图片规划。
2. AI 生成头图并自动打开预览，确认后继续。
3. AI 生成详情页切片 01 并自动打开预览，用于确认详情页方向。
4. 切片 01 确认后，AI 连续生成切片 02～06，中途不逐张打开预览。
5. 六张切片完成后，AI 拼接并打开完整详情页预览，同时告知文件位置。
6. 完整详情页确认后，AI 会主动询问商品名称、商品编码和特殊命名要求。

最终会得到：

```text
商品名称_头图.png
商品名称_完整详情页.png
详情页切片01.png ～ 详情页切片06.png
```

没有特殊命名要求时，最终文件夹默认命名为：

```text
商品编码_商品名称
```

## 注意

- 商品外观、Logo、包装文字和配件必须与实拍图一致。
- 不编造尺寸、材质、容量、成分、认证或医疗功效。
- 有真实操作动作的商品，要写清楚方向、手持位置和受力位置。
