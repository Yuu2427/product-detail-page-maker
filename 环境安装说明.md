# 环境安装问题

正常情况下，请让 Codex 或 Trae Solo 自动检查和配置环境，不需要手动操作。

如果安装失败，将完整报错发送给 Codex 或 Trae Solo，并输入：

```text
请分析这个安装报错并完成修复。
修复后运行本 Skill 的 scripts/check_environment.py。
只有环境检查通过后，才告诉我安装成功。
```

## 常见问题

### `spawn git ENOENT`

电脑没有 Git。建议改用 README 中的“Trae Solo + GitHub”或“本地文件夹 + Trae Solo”。

### 找不到 Python

让 Codex 或 Trae Solo 安装 Python 3.10 以上版本，然后重新执行环境检查。

### `No module named PIL`

让 Codex 或 Trae Solo 根据 `requirements.txt` 安装依赖，然后重新执行环境检查。

### Skill 没有被识别

重启 Codex 或 Trae Solo。如果仍未识别，让 AI 检查安装目录中是否直接存在 `SKILL.md`。
